# Bootstrap 3 auto Direction | generate RTL & LTR CSS with SASS
Grunt for task runner
## Installation
    git clone git@github.com:miladbruce/bootstrap-sass-rtl-ltr.git
    npm install
    
## Usage
    
    grunt  //build dist files
    grunt watch // for watching
